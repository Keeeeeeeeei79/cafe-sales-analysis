# cafe-sales-analysis

## Overview
Analysis of 12 months of cafe sales data (34,842 transactions) to identify revenue trends and business insights.

## Key Findings
- Annual revenue: NZD 242,915
- Top selling item: Toastie (NZD 30,968)
- Peak hour: 9am
- Weekday revenue is 40% higher than weekends

## Tools Used
- Microsoft Excel (SUMIF, Charts, Dashboard)

## Skills Demonstrated
- Data cleaning and aggregation
- Business insight generation
- Data visualisation
